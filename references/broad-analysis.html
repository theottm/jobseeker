1 Problem description

What do I want to achieve with this tool ?

1.1 Definitions

1.1.1 What is the Target Data ?

I want to have access to the key data that make a job or a carreer path interesting. Here is a non-exhaustive list.

  1. is it easy to access ?
    1. is it new ?
    2. what formation is required ?
    3. what is the entrance cost ?
    4. what success stories ?
  2. demand > supply
  3. loan
  4. re-usability
    1. tremplin
    2. multiple use case
  5. connection with your parcours

1.1.2 What problem sould this tool solve ?

what is it you want to solve / answer ?

  1. can there be a tool to help me find a job / seek for opportunities ?
    1. can this be quicker the reading / regular digging ?

      What this tool could/should bring :

      1. have an overview of field to dig further in
      2. needs to be quick to browse : efficient summary
      3. needs to be reliable : data quality / second check

1.2 Objectives

1.2.1 Needs

  1. need to find a job quickly
  2. need to be able to have a market overview
  3. need to be able to propose a data product
    1. for my portfolio
    2. as an open source webapp

1.2.2 Will

  1. want to have alerts for job opportunities
  2. want to reveal career paths / market trends
  3. want to keep track of my interests to have suggestions of paths

1.2.3 Use

  1. target opportunities
    1. sheets of wanted words
    2. query matching algorithms
  2. data exploration
  3. cluster
    1. nlp
  4. find jobs I didn't know about
  5. get warned if new opportunities
  6. use it as a model for finding my perfect match in the world / exploring the economy
  7. make it open source and useable by anyone

1.2.4 Features

  1. Update
  2. Clustering
  3. Visualization

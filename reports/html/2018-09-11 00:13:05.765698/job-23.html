Job number 23---------------------------------------------------------------------------------------------------- Full Stack Developer, with frontend focus Arweave ----------------------------------------------------------------------------------------------------

Who we are

Arweave’s mission is to create a global, immutable hard drive on a new type of blockchain. On this new, immutable, hard drive we intend to build an uncensorable archive of human knowledge and history.

We’re an expanding startup and we need extra hands on deck. We’re building products in-house like our Chrome extension crypto wallet, as well as working with external partners to plan and deliver products, and building technical tools to support the rest of the team. We’ll need someone who is flexible and happy to work across disciplines and projects - sometimes you’ll be designing and building a website using frontend tech, sometimes we’ll need tools building to help monitor our network using some backend tech hosted on AWS or similar. We’re a small team doing big things, so we go fast and break things.

What you’ll do

You’ll be located in our Berlin office and work closely with the core team, but should be able to work independently and be self-directed.

Primarily designing, developing, and testing Javascript based applications that work with the Arweave immutable storage network. The role is flexible and you’ll be given freedom to give your input and proactively shape our products and processes. You’ll be involved in all aspects of the development cycle from initial ideas, gathering requirements, prototyping, development, deployment and maintenance.

The Arweave network is accessible over a JSON based HTTP API which lets us develop applications using traditional web technologies, so you’ll need to be experienced with core frontend technologies like JS, HTML, CSS, have experience with some backend languages e.g. Python, PHP, Node, Go, etc, database technologies like MySQL, and have general *nix server admin skills. Knowledge about AWS and cloud services would be useful.

As our current and planned products are mainly Javascript SPAs you’ll need experience with:

  • Strong practical knowledge of ES6
  • Practical experience of using Vue.js, React, Angular, or similar JS frameworks along with Vuex/Redux or some other similar state management pattern
  • SASS/SCSS or LESS
  • Webpack, Gulp, Grunt or similar build tools and preprocessors
  • Git, testing and CI tools

We aim to open source as much as possible, so attention to detail is important and you’ll be responsible for ensuring code is of high quality and well documented.

An eye for design is needed as you’ll be building and shaping user interfaces. Some understanding of blockchains would help but this can be learnt on the job.

The role will be based in our office in Berlin. We are looking to fill the position immediately.

To apply, and if you have any questions, feel free to reach out to us via email at kyle [at] arweave.org.

About Us

By providing permanent, low-cost, and truly censorship-free data storage for everyone, we strive to democratise the information space. To achieve this, our team has built and now maintains an online, open source, blockchain-based network providing decentralised data storage distributed worldwide.
We are an English-language team of very skilled and highly motivated professionals and entrepreneurs with diverse academic and professional backgrounds.
In spring 2018, we graduated from the Techstars Berlin accelerator programme and have access to a worldwide network of expert advisors and investors.
We’ve raised a total of $8.7 million in several funding rounds from institutional investors and the wider community via a token crowdsale.
Arweave is an equal opportunity employer and welcomes applications from everybody.
